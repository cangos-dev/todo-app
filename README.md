# Todo App

My first JavaScript project — простой менеджер задач для управления повседневными делами.

---

## Возможности

- Добавление задач  
- Удаление задач  
- Отметка выполненных задач  
- Редактирование задач  
- Категории задач  
- Теги через `#`  
- Дедлайны  
- Подсветка просроченных задач  
- Поиск задач  
- Фильтры (все / активные / выполненные)  
- Drag & Drop (изменение порядка задач)  
- Undo / Redo  
- Темная тема  
- Статистика задач  

---

## Технологии

- HTML  
- CSS  
- JavaScript  
- LocalStorage  

---

## Как запустить

1. Скачайте проект  
2. Откройте файл `index.html` в браузере  

---

## Как работает

- Все задачи сохраняются в `localStorage`  
- Состояние обновляется в реальном времени  
- Поддерживается история изменений (undo / redo)  
- Drag & Drop позволяет менять порядок задач  
- Поиск и фильтры помогают находить задачи
