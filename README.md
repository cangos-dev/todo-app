# Todo App

Task manager web application for managing daily tasks.  
Простое приложение для управления задачами, написанное на JavaScript.

---

## 🚀 Description | Описание

Todo App allows you to conveniently manage your task list: create, edit, delete and track tasks.  
Todo App позволяет удобно управлять списком задач: создавать, редактировать, удалять и отслеживать их выполнение.

---

## ✨ Features | Возможности

- Add and delete tasks  
  Добавление и удаление задач  

- Edit tasks  
  Редактирование задач  

- Mark tasks as completed  
  Отметка выполненных задач  

- Categories and color labels  
  Категории и цветовые метки  

- Tag support (#)  
  Поддержка тегов (#)  

- Task deadlines  
  Установка дедлайнов  

- Highlight overdue tasks  
  Подсветка просроченных задач  

- Search tasks  
  Поиск по задачам  

- Filtering (all / active / completed)  
  Фильтрация (все / активные / выполненные)  

- Drag & Drop sorting  
  Drag & Drop для изменения порядка  

- Undo / Redo  
  Отмена и возврат действий  

- Dark theme  
  Темная тема  

- Local storage support  
  Сохранение данных в localStorage  

---

## ⚙️ How it works | Как работает

- Tasks are saved in localStorage  
  Все задачи сохраняются в localStorage  

- UI updates after each change  
  Интерфейс обновляется при каждом изменении  

- Undo / Redo history is supported  
  Поддерживается история изменений (Undo / Redo)  

- Drag & Drop changes task order  
  Drag & Drop изменяет порядок задач  

- Search and filters help find tasks quickly  
  Поиск и фильтры помогают быстро находить задачи
