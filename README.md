# Todo App

Простое приложение для управления задачами, написанное на JavaScript.

#JavaScript #Frontend #TodoApp #VanillaJS #LocalStorage #Portfolio

---

## Описание

Todo App позволяет удобно управлять списком задач:
создавать, редактировать, удалять и отслеживать их выполнение.

---

## Возможности

- Добавление и удаление задач  
- Редактирование задач  
- Отметка выполненных задач  
- Категории и цветовые метки  
- Поддержка тегов (`#`)  
- Установка дедлайнов  
- Подсветка просроченных задач  
- Поиск по задачам  
- Фильтрация (все / активные / выполненные)  
- Drag & Drop для изменения порядка  
- Undo / Redo  
- Темная тема  
- Сохранение данных в `localStorage`  

---

## Технологии

- HTML  
- CSS  
- JavaScript  
- LocalStorage  

---

## Как запустить

1. Склонируйте репозиторий  
2. Откройте файл `index.html` в браузере  

---

## Как работает

- Все задачи сохраняются в `localStorage`  
- Интерфейс обновляется при каждом изменении состояния  
- Поддерживается история изменений (Undo / Redo)  
- Drag & Drop изменяет порядок задач в списке  
- Поиск и фильтры помогают быстро находить задачи
