# Todo App

Простое приложение для управления задачами.

## 🚀 Возможности

- Добавление задач
- Удаление задач
- Отметка выполненных задач
- Сохранение задач в браузере (localStorage)

## 💻 Технологии

- HTML
- CSS
- JavaScript
